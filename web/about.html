DreamBox Speller

Built with go version go1.14.4 darwin/amd64 and JQuery 3.5.1 If you are seeing this assume you have followed README.md instructions to build and run

Avalible HTTP API

Spelling API Speller Spelling method possible status codes

  • 400 - bad user input or missing query, empty body
  • 404 - no results no suggestions, json returned in body
  • 200 - exact match or suggestions, json returned in body
Spelling method JSON format
{
ExactMatch [true/false],
UserInput string,
Suggestions []string,
Repeating [true/false],
MissingVowels [true/false],
MixedCase [true/false],
}
