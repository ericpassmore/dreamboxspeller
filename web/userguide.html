Type in a word and hit GO! The last 10 results appear below the query window.

To clear previous results type :clear into the input field and hit go

To restore default color theme type :resetcolor into input field and hit go

Options

Relax Y. Y is mostly a vowel, sometimes a consonant, and sometimes something in between. By default the service will not show words with Y consonants unless they are in the user's input. Selecting the Relax Y option relaxes this constraint and shows suggestions with Y regardless of the user input having a Y Try this out with example of "ellow" with and without the option.

Order Not Important When looking for spelling suggestions is the order of letters of important? By default order is important. For example a user input of "spam" returns no results. Selecting Order Not Important will return more results because the service will look through the dictionary for matching letters regardless of order. Try this option on with the user input of "spam" and you will see more suggestions. For example "maps"

Debug this option is not avalible via the web interface. We assume the web user does not have access to the filesytem and the log file. To use this option add "&debug" to the end of the spelling url. Example http://localhost:8080/spelling?q=example&debug

Dark Mode

Toggle switch to change colors between light and dark modes. Choice is preserved by saving to browser's local storage. For this reason the :resetcolor command is provided to clear local storage setting, and reset the user interface to default settings.
